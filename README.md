# Shopper Spectrum: Customer Segmentation and Product Recommendation in E-Commerce

## Project Overview
This project analyzes online retail transaction data to identify customer segments using RFM analysis and K-Means clustering while generating product recommendations using Item-Based Collaborative Filtering.

## Business Objectives
- Customer Segmentation
- Personalized Product Recommendation
- Customer Retention
- Revenue Optimization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Seaborn
- Matplotlib
- Streamlit

## Machine Learning Models
1. K-Means Clustering
2. Agglomerative Clustering
3. Item-Based Collaborative Filtering

## Features
- RFM Analysis
- Customer Segmentation
- Product Recommendation
